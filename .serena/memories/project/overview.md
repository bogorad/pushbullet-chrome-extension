Pushbullet Chrome Extension is an unofficial Manifest V3 compatible Pushbullet client for Chrome. Tech stack: TypeScript, Chrome extension APIs, esbuild bundling, Vitest, ESLint. Main user-facing surfaces: popup, options page, debug dashboard, notification detail, and background service worker. Core app areas live in src/background, src/popup, src/options, src/app, src/infrastructure, src/lib, and src/types. Architecture notes from repo AGENTS: repository pattern, event bus, state machine, single-flight/session initialization patterns, strict TypeScript, graceful error handling, security checks on messaging, and CSP-safe extension behavior.
