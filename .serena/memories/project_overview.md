Pushbullet Chrome Extension is an unofficial Manifest V3 Pushbullet client for Chrome built with TypeScript, Vue-less DOM UI, Vitest, ESLint, and esbuild. Main areas: src/background for service worker lifecycle and message handling, src/popup for popup UI, src/app/session for session cache/orchestration, src/infrastructure/storage for storage repository and IndexedDB cache, tests/ for Vitest suites.
